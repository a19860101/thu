Table 表格排版

  • 軟體輔助
  • 開發時間較短
  • 修改時間又臭又長
  • 不符合標準
  • 適合不需修改的活動頁

Float 浮動

  • 標準
  • 相容性高
  • 開發時間較長
  • 修改時間較短
  • 浮動飄來飄去很煩
  • 適合所有網站

Flexbox 彈性盒子

  • 標準
  • 相容性較低
  • 開發時間更短
  • 沒有浮動問題
  • 適合所有網站

浮動

飄來飄去到底是要飄去哪？

問題

只要是浮動元素，他的父元素與相鄰元素都會發生問題

解決方案

  • height
  • clear
  • overflow
  • clearfix

height

將消失的高度補回去

height:60px;

clear

清除對於浮動元素的影響

clear:both;

overflow

超出範圍自動補滿

overflow:auto;

clearfix

浮動修正

                .clearfix::after {
                        content:"";
                        display:block;
                        clear:both;
                    }
